My setup for any data science project with Sublime, project manager, cookiecutter

Prerequisites: Installed sublime and added sublime.exe to path. Added package manager and project manager to Sublime. For pre-build folder structure install cookiecutter.

TL;DR

Follow these steps to setup your project:

  1. Create a project directory with a meaningful name and open it
  2. Click the address bar or press Alt+d, type cmd and hit enter
  3. Run subl . to open Sublime for the current directory
  4. Create a new project with Ctrl + shift + P and run Project Manager: Add new Project
  5. Name the project
